# Glossary

## Terms
- **Core API**: The backend API surface implemented via Next.js route handlers under `Backend/core-api`.
- **Merchant Admin**: The main merchant dashboard application under `Frontend/merchant-admin`.
- **Ops Console**: Internal operations console under `Frontend/ops-console`.
- **Storefront**: Customer-facing storefront under `Frontend/storefront`.
- **Marketing**: Public marketing site under `Frontend/marketing`.
- **Prisma / @vayva/db**: Database schema + generated client in `infra/db`, exported to the monorepo as `@vayva/db`.
- **VPS 1**: App server (Docker) used for components that are not on Vercel.
- **VPS 2**: Database server (PostgreSQL) used for production databases.
