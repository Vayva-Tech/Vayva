# Glossary

**Owner:** Nyamsi Fredrick, Founder  
**Last Updated:** March 2026

---

## A

**AI Order Capture**
: Automated extraction of order details from WhatsApp/Instagram conversations using AI.

**API (Application Programming Interface)**
: Interface that allows different software applications to communicate.

## B

**BullMQ**
: Redis-based queue system used for background job processing.

**Better Auth**
: Authentication library used for user session management.

## C

**Core API**
: Vayva's main backend API built with Fastify.

**Customer**
: A person who purchases from a merchant on Vayva.

## D

**Dashboard**
: The merchant admin interface for managing business operations.

**Delivery**
: The process of transporting orders to customers.

## E

**Evolution API**
: WhatsApp Business API gateway used for WhatsApp integration.

**E2E (End-to-End)**
: Testing that simulates real user scenarios.

## F

**Fastify**
: High-performance Node.js web framework used for Core API.

**Fulfillment**
: The complete process from order confirmation to delivery.

## G

**Groq**
: AI/LLM service provider used for AI features.

## H

**Hetzner**
: VPS hosting provider for backend services.

## I

**Inventory**
: The quantity of products available for sale.

## J

**Job Queue**
: System for processing background tasks asynchronously.

## K

**Kwik Delivery**
: Last-mile delivery service integrated with Vayva.

## L

**LCP (Largest Contentful Paint)**
: Web performance metric measuring loading speed.

## M

**Merchant**
: A business using Vayva to sell products/services.

**Merchant Admin**
: Dashboard application for merchants to manage their business.

**Monorepo**
: Single repository containing multiple related projects.

## N

**NDPR (Nigeria Data Protection Regulation)**
: Nigerian data privacy law Vayva complies with.

**Neon**
: Serverless PostgreSQL provider used for database hosting.

## O

**Ops Console**
: Internal dashboard for platform operations and support.

**Order**
: A customer's purchase request.

## P

**Paystack**
: Payment processor integrated for Nigerian payments.

**pnpm**
: Package manager used in the Vayva monorepo.

**Prisma**
: ORM (Object-Relational Mapping) tool for database access.

**Product**
: An item available for sale in a store.

## Q

**Queue**
: System for managing background jobs.

## R

**Redis**
: In-memory data store used for caching and queues.

**Rescue (Vayva Rescue)**
: AI-powered incident response system.

## S

**SalesAgent**
: AI component that handles WhatsApp order conversations.

**Storefront**
: Customer-facing website where products are sold.

## T

**Template**
: Pre-designed store layout that merchants can customize.

**TTFB (Time to First Byte)**
: Web performance metric measuring server response time.

## U

**Upstash**
: Managed Redis service provider.

## V

**Vayva**
: The commerce platform for African businesses.

**Vercel**
: Frontend hosting platform used for Next.js applications.

## W

**WhatsApp Commerce**
: Selling products through WhatsApp conversations.

**Worker**
: Background job processor for async tasks.
