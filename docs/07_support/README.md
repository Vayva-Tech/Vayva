## Common Issues

### Account Issues

**Can't log in:**
1. Check email/password
2. Reset password
3. Check email spam
4. Contact support

**Account suspended:**
1. Check email for reason
2. Review terms of service
3. Submit appeal if needed

### Payment Issues

**Payments not processing:**
1. Check Paystack status
2. Verify API keys
3. Check webhook configuration
4. Contact support

**Withdrawal issues:**
1. Verify bank details
2. Check minimum amount
3. Review withdrawal history
4. Contact support

### Technical Issues

**Store not loading:**
1. Check custom domain DNS
2. Verify SSL certificate
3. Clear browser cache
4. Contact support

**WhatsApp not working:**
1. Check Evolution API status
2. Verify WhatsApp Business connection
3. Check phone number
4. Contact support

## Escalation

### When to Escalate

- Issue not resolved in SLA
- Critical business impact
- Security concern
- Repeated issue

### Escalation Path

```
L1 Support → L2 Technical → Engineering → Leadership
  (4h)         (24h)          (48h)       (As needed)
```
