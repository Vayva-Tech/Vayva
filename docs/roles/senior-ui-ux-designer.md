# Senior UI/UX Designer - Role Documentation

## 🎨 **POSITION OVERVIEW**
**Reports to:** Head of Product/Design Director  
**Team Collaboration:** Works closely with Lead Engineer and Frontend Developers  
**Project:** Vayva Multi-Industry Dashboard Platform  

## 🎯 **PRIMARY RESPONSIBILITIES**

### **Design System Leadership**
- Create and maintain unified design system across 22 industry verticals
- Develop industry-specific themes while maintaining brand consistency
- Design responsive interfaces that work seamlessly across desktop, tablet, and mobile
- Establish design tokens, component libraries, and style guides

### **User Experience Optimization**
- Conduct user research and usability testing for each industry template
- Create user journey maps and information architecture
- Design intuitive workflows for complex business operations
- Optimize dashboard layouts for different user personas

### **Visual Design Excellence**
- Craft pixel-perfect interfaces with modern aesthetic
- Design data visualization components for complex analytics
- Create engaging onboarding experiences for new users
- Develop brand-consistent marketing materials and promotional assets

## 🎨 **DESIGN REQUIREMENTS**

### **Industry Expertise Needed**
- **Retail/Fashion/E-commerce:** Inventory management, POS systems, visual merchandising
- **Healthcare/Medical:** HIPAA-compliant interfaces, patient data visualization
- **Legal/Professional Services:** Document management, case tracking, billing systems
- **Creative/Portfolio:** Gallery displays, project showcases, client collaboration tools
- **Education/LMS:** Course management, student tracking, interactive learning interfaces

### **Technical Design Skills**
- **Design Tools:** Figma (expert), Adobe Creative Suite, Sketch
- **Prototyping:** InVision, Marvel, Principle for interactive prototypes
- **Frontend Awareness:** Understanding of React components, CSS frameworks
- **Accessibility:** WCAG 2.1 compliance, inclusive design principles

### **Design System Components**
- **Core Elements:** Color palettes, typography scales, spacing systems
- **Interactive Components:** Buttons, forms, navigation, data tables
- **Industry Widgets:** Charts, graphs, KPI displays, progress indicators
- **Notification Systems:** Alerts, toasts, banners, modal dialogs
