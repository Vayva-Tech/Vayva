# CourseAcademy Template

A modern, conversion-focused online course platform for individual educators and small academies.

## Pages Built
- `/` - Landing page with hero, featured courses, testimonials, FAQ
- `/courses` - Course catalog with filtering
- `/course/[id]` - Course detail with curriculum
- `/about` - About the instructor/academy
- `/pricing` - Simple pricing plans
- `/contact` - Contact form
- `/blog` - Educational blog
- `/auth/*` - Login/signup

## Multi-Tenant Ready
- Configurable branding
- Webstudio editable sections
- Modular components
