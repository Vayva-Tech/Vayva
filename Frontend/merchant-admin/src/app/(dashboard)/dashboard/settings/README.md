# Industry-Specific Settings Directory

This directory contains settings pages that are unique to specific industries and cannot be unified.

## Structure

Each industry folder contains:
- Workflow-specific configurations
- Industry compliance settings
- Specialized feature toggles

## Industries with Specific Settings

- `/beauty` - Beauty industry workflows and configurations
- `/grocery` - Grocery industry workflows and configurations  
- `/real-estate` - Real estate industry workflows and configurations
- `/nonprofit` - Nonprofit industry workflows and configurations
- `/creative` - Creative industry workflows and configurations

## Common Settings (Moved to Unified Panel)

The following settings have been moved to the unified settings panel accessible via the Settings button in the header:

- Profile settings
- Store/branding settings
- Team and roles
- Payments and billing
- Shipping configuration
- Notifications
- Integrations
- Navigation
- AI Agent configuration

## Access

Industry-specific settings are accessed through industry dashboards or direct navigation.
Common settings are accessed through the unified Settings panel in the header.
