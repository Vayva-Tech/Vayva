## 📝 LESSONS LEARNED

### Root Causes Identified:
1. **File Extensions Matter**: TypeScript strictly enforces `.tsx` for JSX
2. **Text Content Traps**: `<18` looks like generic syntax to TS parser
3. **Indentation Sensitivity**: Multi-line `.map()` returns need precise formatting
4. **Cache Sensitivity**: Stale build artifacts can cause phantom errors
5. **Parser Recovery**: One file's errors can cascade to others

### Prevention Strategies:
1. Use `.tsx` for any file with JSX - no exceptions
2. Spell out comparisons in text ("under 18" not "<18")
3. Use explicit `return` with curly braces for complex maps
4. Regular cache cleaning in CI/CD pipeline
5. Isolate errors quickly to find root causes
